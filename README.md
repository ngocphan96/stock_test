# Radar danh mục — demo khảo sát nhu cầu

Bản demo **tĩnh** (không cần server) để gửi link cho khách hàng tiềm năng tham khảo và đo nhu cầu **trước khi** xây sản phẩm thật.

> ⚠️ Mọi số liệu trong demo là **minh hoạ**, không phải khuyến nghị đầu tư.

## Link demo
Sau khi bật GitHub Pages, dán link vào đây:
**https://<tên-github>.github.io/radar-demo/**

- Trang giới thiệu: `index.html`
- Xem tất cả màn hình: `menu.html`

## Các màn hình (14 trang)
**Lõi:** index · onboarding · radar · detail · zalo
**Niềm tin:** ngay-binh-thuong · track-record · canh-bao
**Test giá:** bang-gia · nganh · ro-tiem-nang · bao-cao-tuan
**Cá nhân hóa:** cai-dat
**Điều hướng:** menu

## Triển khai
Xem hướng dẫn từng bước trong **`HUONG-DAN.md`** (deploy GitHub Pages, nối form Formspree, gắn tên miền, quyền riêng tư).

## Cấu trúc
Mỗi trang là một file `.html` **tự chứa** (CSS/JS nằm trong file) — mở lẻ vẫn chạy, không phụ thuộc file khác. Liên kết giữa các trang dùng đường dẫn tương đối nên hoạt động tốt trên GitHub Pages.
